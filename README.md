# Contact-Management

## Используемы технологии:

- TypeScript
- Composition Api
- VueUse(useStorage, useDebounceFn, useAsyncValidator)
- async-validator
- Tailwind

## Установка

```sh
npm install

yarn
```

### Запуск

```sh
npm run dev

yarn dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build

yarn build
```
